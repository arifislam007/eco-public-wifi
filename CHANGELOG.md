# Changelog

## [1.0.0] - Initial Release

### Features
- FreeRADIUS 3.x integration with MySQL backend
- Responsive Bootstrap-based captive portal
- Admin panel for user/voucher management
- User authentication with username/password
- Session timeout support
- Account expiry dates
- Usage tracking and logging
- Online user monitoring
- Rate limiting for login attempts
- Security features (SQL injection protection, password hashing)
- Support for MikroTik, OpenWRT, and UniFi routers

### Components
- Database schema (radcheck, radreply, radacct tables)
- FreeRADIUS configuration files
- PHP captive portal (frontend + backend)
- Admin panel with dashboard
- CLI scripts for user management

### Documentation
- README.md with installation guide
- INSTALL.md with step-by-step instructions
- Configuration examples for routers
